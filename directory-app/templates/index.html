Internal Directory Service

This is an internal record-keeping tool. It is intentionally not linked from any public page or sitemap. Access requires an admin token sent via the X-Admin-Token header on API requests.

  • GET /admin/employees — list records
  • POST /admin/employees — add a record
  • DELETE /admin/employees/<id> — remove a record
  • GET /route?display_email=...&anchor=... — resolve routing target
